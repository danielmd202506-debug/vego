# VEGO Presentation Materials

This folder contains shareable presentation files for the VEGO design-system work.

## Files

| File | Use |
|---|---|
| `VEGO-Design-System-Leadership-12p-v7.pptx` | Condensed solution narrative for leadership review |

## Notes

- These files are copied out of the local `outputs/` folder so they can be versioned and shared from GitHub.
- `outputs/` remains ignored because it contains generated drafts and intermediate files.
- For the latest engineering source, use `vego-design-system-foundation-v1/`.
